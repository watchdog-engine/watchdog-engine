# Roadmap

## ✅ v0.1 — FIRST LIGHT *(shipped)*
Proof of concept on real shelter footage: detection, basic overlay, live
virtual-camera output. Proved the idea works on ordinary hardware.

## ✅ v0.2 — TECHNICAL *(shipped)*
Full operator console: event log, activity pulse, alert engine, camera-motion
handling, and the self-healing runtime layers.

## ✅ v0.3 — SENTINEL *(current build — shown in live streams)*
Perception overhaul:
- Night-IR-tuned detection stack, validated frame-by-frame against real footage
- Persistent identity tracking with long-hold policy for sleeping animals
- Sub-pixel ego-motion compensation — identities survive full camera pans
- Motion sentinel: <1 s lock-on for new movement
- Stillness continuity through pans and off-frame periods
- 32-minute unattended soak test: zero faults, zero manual repairs

## 🔜 v0.9 — FIELD KIT *(funded by launch)*
- Direct RTSP/ONVIF-class ingestion from off-the-shelf security cameras
- Multi-room sessions
- Remote alert delivery (mobile push / messaging)
- Per-facility threshold configuration

## 🔜 v1.0 — DEPLOYMENT *(applications open)*
- First real installations — free for participating shelters
- Home-care program for sick and disabled dogs
- Application program opens the day v1.0 ships
